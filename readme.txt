- Los modelos presentados son:
    -Clientes - corresponde a los clientes de la empresa
    -Maritimas - son las proveedoras de tucci, quien les paga para mover las mercancias de sus clientes
    -Rutas - son las rutas que hacen las mercancias. 
    -Precios - son los precios de venta para cada ruta. 
    -Busqueda - para filtrar clientes o maritimas

- Home : localhost:8000/aplicacion - desde aqui podemos acceder a traves de la barra superior a las diferentes consultas,
ademas se incluyen las mejores opiniones de los clientes de la semana para motivar al equipo!.

- Acceso a las consultas por link: Dentro de cada consulta tenemos las opciones para redirigirnos tambien a Create, Update o Delete. 

    - Consultar clientes: localhost:8000/aplicacion/clientes
    - Consultar maritimas: localhost:8000/aplicacion/maritimas
    - Consultar rutas: localhost:8000/aplicacion/rutas
    - Consultar precios: localhost:8000/aplicacion/precios
